Mon Projet – Application Web Météo


Description du projet:
-Mon Projet est une application web qui permet de voir la météo dans plusieurs villes partout dans le monde.

Technologies utilisées:
-HTML pour la structure de la page
-CSS pour le design, animations.
-JavaScript pour la logique dynamique, manipulation du DOM et intégration API.
-API OpenWeather pour la récupération des données météo.0
-Fonts Google pour la typographie.

Fonctionnalités principales:
-Page web sobre simplistique.
-Pas trop chargé ni moderne.
-Simple à comprendre et à prendre en main.
-Capable de donner la météo de plusieurs villes en même temps sous forme de cartes et aussi capable de supprimer ces cartes.

Lien vers la page GitHub Pages:
-https://malekdhaouadi1.github.io/Projet-Dev-Web/

Nouveautés explorées:
-Conception de composants UI réutilisables et dynamiques.
-Manipulation avancée du DOM avec JavaScript.
-Utilisation d’une API externe (OpenWeather).
-Gestion d’événements sur des éléments créés dynamiquement (bouton supprimer, touche Enter).

Difficultés rencontrées:
-Gestion des erreurs API et affichage UX correct.
-Ajouter et supprimer dynamiquement des cartes tout en conservant un design propre.
-Optimiser l’affichage pour différents écrans et tailles de fenêtres.

Solutions apportées:
-Gestion des erreurs API via catch et affichage d’un message utilisateur clair.
-Ajout d’événements dynamiques pour chaque carte (supprimer, Enter).
-Tests et ajustements sur plusieurs navigateurs et tailles d’écran.
-Application progressive des styles CSS modernes pour améliorer l’UX et le design global.
-Et un tout petite peu d'aide de la part de ChatGPT.
